How to Get Highlighted Text and Display Correct Video

Python Source:

-Get selected text
-Change Launcher.html var keyword using FILE IO to contain the selected text
-Call Launcher.html, which will bring up another page (pop up video) and set the query variable 

Launcher.html
-Contains var keyword
-Call javascript to open up the video pop up page with query = keyword and then close itself.
	***Use this code***: 
	window.open('ASLVideo.html?keyword='+keyword, '_self').close();

ASLVideo.html
-Get query word using index.html functionality
	-If no query, display No Video Found and then close self after 3 seconds
	-Otherwise, display the video received from Youtube with the keyword
